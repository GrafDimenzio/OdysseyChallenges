## NoBonk
Resets every time any time a players bonks.

## Config
- ResetOnlyBonkingPlayer: when enabled only the player that bonked will be reset
- ResetAllToBonkingPlayer: when enabled every player will be teleported to the same stage as the player that bonked
- ResetOnlyPlayersOnSameStage: when enabled all players on the same stage where the bonk happened will be reset (ResetOnlyBonkingPlayer mus tbe disabled)
- ResetToKingdom: when enabled the stage that will be redirected to will be the kingdom stage so you will be booted out of any sub area
